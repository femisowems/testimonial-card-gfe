# Testimonial Card

A clean, responsive testimonial card component built with React and Tailwind CSS. This project demonstrates a modern card layout that adapts to different screen sizes, handling both single and multiple testimonials gracefully.

## Features

- **Responsive Design**: Adapts seamlessly from mobile to desktop.
- **Dynamic Layout**: 
  - Centers the card when displaying a single testimonial.
  - Switches to a grid layout for multiple testimonials.
- **Modern Styling**: Styled with Tailwind CSS for a professional look.

## Built With

- [React](https://react.dev/) - UI Library
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Vite](https://vitejs.dev/) - Frontend Tooling

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/femisowemimo/testimonial-card.git
   ```
2. Navigate to the project directory:
   ```bash
   cd testimonial-card
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Usage

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the project.
